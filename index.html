  • About This Project

    This is the Cloud Resume Challenge authored by Forrest Brazeal. It tasks the user with creating and deploying an online CV backed by various cloud-based technologies. I chose to use AWS and worked with the following technologies:

    • S3 - for the frontend hosting

    • CloudFront - for efficient content distribution

    • Route 53 - for the domain name registration

    • Certificate Manager - for securing the site with HTTPS

    • DynamoDB - for storing the web page views

    • Lambda - for creating the API between the frontend javascript and the backend database

    Finally, as an introduction to DevOps, I was able to implement CI/CD (Continuous Integration and Continuous Deployment) using Github Actions with some pipeline configurations in YAML so that the live site is updated on every push. You can find more about my experience building this cloud CV here.
